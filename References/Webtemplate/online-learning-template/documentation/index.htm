About the template

Bromine ecommerce template for every type of website, including 20 different page layouts to build your own website.

Bromine HTML5 Template can be used for multi-purpose websites. Wether you want to create a ecommerce shop, a business website, a blog or even a personal portfolio, it is always the right solution. Bromine is a template for fashion shops and ecommerce websites. It is a highly suitable template for all types of ecommerce. It has flat and purpose oriented design, responsive layout and special features like appointment forms, as well as blog layouts and landing pages. Use easy template options and shortcodes to customize your website. This template is powered by Twitter Bootstrap v3 (the most popular front-end framework). It means that it will be easier for you to do the changes and to increase its possibilities, thanks to many plugins it possesses.

Installation

Bromine - Online Learning Platform HTML5 template does not need a real installation. Just upload all files included in folder ./bromine-html to your server.

Upload Template files

You have two ways to upload Groham Template files:

  1. Upload Bromine files in folder ./marlboro to your server via FTP
    • Unpack the Template .zip file (bromine-html*.zip) you have downloaded.
    • Upload the /bromine folder on your server.

Files

Bromine Template includes many files, within which you'll find the following directories and files, logically grouping common resources and providing both finding and changes.

You will find 3 folders in your .zip file.

Including a documentation, the HTML template and a PSD files folder containing additional .PSD files, you can be sure that everything is completed.

Documentation

Including four different folders with coded stuff and one index.htm for you to open. Just click on the index.html and open it with any browser. Now you can see the documentation in your browser.

JavaScript

All JS files are in assets/ directory.

Template Configuration

Please go to scripts/common.js, and configure the template:

The Required attributes are

  • headerSliderTimeout
  • scrollDownSpeed
  • lightboxMaxWidth
  • defaultSliderConfig.autoplay
  • defaultSliderConfig.autoplaySpeed
  • defaultSliderConfig.infinite
  • defaultSliderConfig.arrows

Sample code is below:


var headerSliderTimeout = 5000
var scrollDownSpeed = 400
var lightboxMaxWidth = 2.5
var lightboxMaxHeight = 1.5

var defaultSliderConfig = {
  autoplay: true,
  autoplaySpeed: 5000,
  infinite: true,
  arrows: false,
}
